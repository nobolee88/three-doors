# Translation Guide

## How to Contribute a Translation

The Three Doors should be readable by anyone on Earth, in their own language.

### What to Translate

The primary document is `index.html` in the root directory. This is the full interactive piece.

### How to Submit

1. Fork this repository
2. Create a new directory under `/translations/` with the ISO 639-1 language code (e.g., `de/` for German, `ja/` for Japanese)
3. Copy `index.html` into your language directory
4. Translate the content — see guidelines below
5. Submit a pull request

### Translation Guidelines

- **Translate meaning, not words.** Cultural adaptation matters more than literal accuracy.
- **Keep all source citations intact.** Links and attribution should remain in their original language.
- **Preserve the structure.** The HTML structure, CSS, and JavaScript should remain unchanged.
- **Add local context where relevant.** If your country has its own military AI programs, mention them in the appropriate section.
- **Maintain the tone.** This is factual, not sensational. No enemies, no propaganda. Pro-human.

### Quality Review

- Priority 1 translations require human review before merging
- Priority 2-3 translations welcome as machine-assisted drafts (clearly labeled)
- All translations should be reviewed by at least one native speaker before being marked as verified
